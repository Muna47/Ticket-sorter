# CRM Ticket Classification Service

A backend API that classifies customer support tickets into predefined categories.

---

## Features

- Express REST API
- Joi Validation
- Helmet Security
- Rate Limiting
- English Support
- Bengali Support
- Structured JSON Responses
- Confidence Scoring
- Human Review Flagging
- Health Check Endpoint

---

## Installation

```bash
git clone <repo-url>

cd ticket-sorter

npm install
```
